üìù Strings - Working with Text

What are Strings?

Strings are sequences of characters used to represent text data like names, messages, or any textual information.

Three Ways to Create Strings:

const doubleQuotes = "Hello World"; const singleQuotes = 'Hello World'; const backTicks = `Hello World`; console.log(doubleQuotes); // "Hello World" console.log(singleQuotes); // 'Hello World' console.log(backTicks); // `Hello World`

üéÆ Try String Operations:

String Concatenation - Old vs New Way:

// Old way with + operator const sentence1 = "Hello " + firstName + " " + lastName + "!"; // Modern way with template literals (backticks) const sentence2 = `Hello ${firstName} ${lastName}!`; // Template literals support expressions const sentence3 = `Hello ${firstName.toUpperCase()} ${lastName}!`;

üî¢ Numbers - Mathematical Operations

JavaScript Numbers

JavaScript has one number type that can represent both integers and decimals (floating-point numbers).

const age = 25; // Integer const price = 19.99; // Decimal const negative = -42; // Negative number const scientific = 1e6; // Scientific notation (1,000,000) // Basic mathematical operations const sum = 10 + 5; // 15 const difference = 10 - 5; // 5 const product = 10 * 5; // 50 const quotient = 10 / 5; // 2 const remainder = 10 % 3; // 1 (modulo) const power = 2 ** 3; // 8 (exponentiation)

‚úÖ Booleans - True or False

Boolean Values

Booleans represent binary logic - something is either true or false. They're essential for making decisions in your code.

const isLoggedIn = true; const hasPermission = false; // Boolean expressions const isAdult = age >= 18; // true if age is 18 or greater const isEqual = 5 === 5; // true (strict equality) const isNotEqual = 5 !== "5"; // true (different types) const isGreater = 10 > 5; // true const isLessOrEqual = 5 <= 10; // true // Logical operators const canAccess = isLoggedIn && hasPermission; // AND const showContent = isLoggedIn || isGuest; // OR const isHidden = !isVisible; // NOT
