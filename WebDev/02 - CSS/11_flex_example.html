Flexbox (Flexible Box Layout) is a powerful CSS layout method that makes it easy to align and distribute space among items in a container. This page demonstrates all the key flexbox properties with interactive examples.

1. Basic Flexbox Container

When you add display: flex to a container, it becomes a flex container and its children become flex items.

ðŸŽ¯ Flexbox Cheat Sheet

Container Properties (apply to parent):

  • display: flex - Creates a flex container
  • flex-direction: row | row-reverse | column | column-reverse
  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly
  • align-items: stretch | flex-start | flex-end | center | baseline
  • flex-wrap: nowrap | wrap | wrap-reverse
  • gap: 20px (space between items)

Item Properties (apply to children):

  • flex-grow: 0 | 1 | 2... (how much to grow)
  • flex-shrink: 1 | 0... (how much to shrink)
  • flex-basis: auto | 200px... (initial size)
  • flex: 1 (shorthand for grow, shrink, basis)
  • align-self: auto | flex-start | flex-end | center | baseline | stretch

Common Use Cases:

  • Navigation bars: justify-content: space-between
  • Centering content: justify-content: center + align-items: center
  • Equal-width columns: flex: 1 on children
  • Responsive layouts: flex-wrap: wrap
  • Sticky footer: flex-direction: column + flex-grow on main content

Pro Tip: Flexbox is perfect for one-dimensional layouts (either row or column). For two-dimensional layouts (rows AND columns), consider CSS Grid!
