üìã Document Structure Overview

Basic HTML Document Structure:

<!DOCTYPE html> <html lang="en"> <head> <!-- All meta information goes here --> </head> <body> <!-- All visible content goes here --> </body> </html>

üß© Essential Meta Tags

<!DOCTYPE html>
DOCTYPE Declaration: Tells the browser this is HTML5. Always put this first!
<html lang="en">
Root Element: Contains all content. The lang attribute helps search engines and screen readers.
<meta charset="UTF-8">
Character Encoding: Allows use of all characters, emojis, and international text. Always use UTF-8!
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Viewport: Essential for mobile responsiveness. Without this, mobile browsers show a zoomed-out desktop view.

üîç SEO Meta Tags

Search Engine Optimization

These meta tags help search engines understand and display your page correctly:

<title>Page Title</title>
Page Title: Appears in browser tab and search results. Keep it under 60 characters.
<meta name="description" content="...">
Meta Description: Shows in search results. Keep it 150-160 characters for best results.
<meta name="keywords" content="...">
Keywords: Less important now, but still used by some search engines.

üì± Social Media Meta Tags

Open Graph (Facebook, LinkedIn, etc.)

<meta property="og:title" content="Your Page Title"> <meta property="og:description" content="Your page description"> <meta property="og:image" content="https://example.com/image.jpg"> <meta property="og:url" content="https://example.com/page"> <meta property="og:type" content="website">

Twitter Cards

<meta name="twitter:card" content="summary_large_image"> <meta name="twitter:title" content="Your Page Title"> <meta name="twitter:description" content="Your description"> <meta name="twitter:image" content="https://example.com/image.jpg">

‚úÖ Best Practices

Always Include These:

  • DOCTYPE declaration
  • UTF-8 charset
  • Viewport meta tag
  • Page title
  • Meta description

Common Mistakes:

  • Forgetting the viewport meta tag (breaks mobile)
  • Not setting charset (causes character display issues)
  • Titles too long (get cut off in search results)
  • Missing meta descriptions (search engines create their own)
